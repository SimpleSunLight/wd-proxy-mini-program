### **小程序**

#### 当前小程序是一个商品代理端、相当于一个批发商
<br/> 

#### 完成功能如下
<br/>   

>1. 验证码登录
>2. 我的个人信息
>3. 购物车
>4. 商品列表
>5. 商品分类
>6. 订单状态
>7. 收件地址
>8. 物流快递信息
>9. 商品详情
>10. 订单详情
>11. 积分列表
